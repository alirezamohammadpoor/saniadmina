# Sania D'Mina — Shopify Theme

Shopify Online Store 2.0 theme for [saniadmina.com](https://www.saniadmina.com/).

Built on [Cadaver 2.0](https://github.com/stefbowerman/cadaver-2.0) boilerplate.

## Development

```
bun run dev
shopify theme dev --store={store_id}
```

## Build

```
bun run build
```
